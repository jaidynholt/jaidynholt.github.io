ScootEZ

2024


ScootEZ is a software engineering project for the course CEN3031 Introduction to Software Engineering, summer 2024.

In response to the challenge of designing a tracking system for an electric scooter rental service, this is a MERN stack web application connected to a server using React.js and Node.js and using a MongoDB database. The application is able to create three different types of accounts, customer, employee, and administrator, which will have their appropriate functionalities according to the client's requirements. It stores the scooter information and account information in a local database.
